# Temperature Converter Project

Цей проект допомагає швидко конвертувати градуси Цельсія у Фаренгейти.

## Як запустити

1. **Переконайтеся, що у вас встановлений Python** (версії 3.14).
2. **Завантажте проект** або скопіюйте файли у папку.
3. **Відкрийте термінал** (або консоль) у папці проекту.
4. **Запустіть скрипт** за допомогою команди:
   ```bash
   python src/main.py
