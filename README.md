# Temperature Converter Project

Цей проект допомагає швидко конвертувати градуси Цельсія у Фаренгейти.

## Як запустити

1. **Переконайтеся, що у вас встановлений Python** (версії 3.14).
2. **Завантажте проект** або скопіюйте файли у папку.
3. **Відкрийте термінал** (або консоль) у папці проекту.
4. **Запустіть скрипт** за допомогою команди:
   ```bash
   python src/main.py

## Команди, які я використовував у цьому проекті

Під час роботи над цим навчальним проектом я опанував такі команди Git:

- `git clone` — скопіювати існуючий репозиторій Git на локальний комп'ютер.
- `git status` — перевірка стану робочої директорії.
- `git add <file>` — додавання файлів до індексу.
- `git commit -m "message"` — створення коміту зі змінами.
- `git reset --soft HEAD~1` — скасування останнього коміту зі збереженням змін у файлах.
- `git checkout -b <branch_name>` — створення нової гілки та перехід на неї.
